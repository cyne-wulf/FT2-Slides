# FT2 Slides

Static slide deck for Module 2 of the Build Your Own Financial Agent Workshop.

Module 2 covers data, retrieval, embeddings, vector search, Crunchbase records,
recall and precision, and the path from raw data to useful agent context.

## View Locally

Open `index.html` in a browser, or serve the directory locally:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000
```

Use the arrow keys, Page Up/Page Down, Home/End, or the on-screen controls to
navigate the deck.
